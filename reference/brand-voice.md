# Brand Voice Guide — kando1合同会社 (@kando1_)

Last updated: 2026-02-18
Sample base: 14 posts (X timeline, Oct–Dec 2025)

---

## 1) Voice Summary

**Positioning voice:**
- インディー感のあるプロダクト開発チーム
- 丁寧だけど堅すぎない
- 「作って→改善して→共有する」ビルドインパブリック型

**Core tone (mix):**
- 70% practical / update-driven
- 20% friendly / human
- 10% playful (軽いノリ、lol、感嘆)

**Language style:**
- 日英ミックス（英語の機能説明 + 日本語の文脈）
- 短文中心、改行でリズムを作る
- 箇条書き（・）を多用

---

## 2) Tone Rules

### Use this tone
- 開発進捗を「事実ベース」で共有
- 読み手の体験価値を先に言う（例: 友達が少なくても遊べる）
- 感謝・お詫びは率直に短く
- 完璧さより、改善中のリアルさを見せる

### Avoid this tone
- 過度に企業的な硬い言い回し
- 誇張マーケ文言（最強、革命、絶対など）
- 長い背景説明（3〜4文超）

---

## 3) Vocabulary & Phrase Bank

### Frequent themes
- update / implemented / added
- beta / user募集中
- strategy / psychology / gameplay
- infrastructure / cost optimization / server

### Signature phrasing patterns
- "We [implemented/added/optimized] ..."
- "Hope you enjoy it!"
- "To [do X]: [step] → [step] → [step]"
- "あなたなら〜？"
- "もちろん無料で遊べます！"
- "β版ユーザー募集中です"

### Emojis / symbols
- 絵文字は少なめ（必要なときだけ）
- 箇条書き記号は `・` を使うとらしさが出る
- 区切りに `→` をよく使う

---

## 4) Sentence Structure

### Typical post shape (preferred)
1. Hook（何をしたか / 何ができるか）
2. 具体（2〜4項目、短い箇条書き）
3. 締め（一言 + CTA or gratitude）
4. Hashtags（2〜4個）

### Length guidance
- 2〜6行が基準
- 1投稿1メッセージ（詰め込みすぎない）
- 改行で読みやすくする

---

## 5) Hashtag Style

Use 2–4 tags from these pools:
- Product: `#meitra` `#明専トランプ` `#明トラ`
- Tech: `#NestJS` `#Flyio` `#CostOptimization`
- Category: `#カードゲーム` `#bridge` `#update`

Rule:
- 毎回同じタグを固定しすぎない
- 技術投稿はTechタグ優先、ゲーム投稿はProduct+Category優先

---

## 6) Brand Personality Sliders

- Formal ↔ Casual: **40 / 60 (slightly casual)**
- Technical ↔ Emotional: **65 / 35 (technical-leaning)**
- Confident ↔ Humble: **55 / 45**
- Playful ↔ Serious: **35 / 65**

## 8) Do / Don’t Checklist (before posting)

- [ ] 1投稿1テーマになっている
- [ ] 読み手メリットが入っている
- [ ] 具体が2〜4個ある（必要時）
- [ ] 文量が長すぎない
- [ ] ハッシュタグは2〜4個
- [ ] トーンが「丁寧 + 親しみ」に収まっている

## 9) Quick Rewrite Examples

### Example input (too corporate)
"本日、当社は新機能を正式リリースいたしました。ユーザー体験の向上に寄与します。"

### On-brand rewrite
We added a new feature to make gameplay smoother.
・Faster room join
・Clearer turn status
・Less waiting time

Hope you enjoy it!
#meitra #明専トランプ #update
