# Item specifics template (Home & Kitchen smalls)

Use these as required fields for each listing:

- Brand: Unbranded
- Type: (match product)
- Material: (from SKU sheet)
- Color: Black / Gray / Mixed
- Number in Pack: (from SKU sheet)
- MPN: Does Not Apply
- Features: Reusable / Easy Clean / Space Saving (match product)
- Condition: New

## Shipping / return policy
- Handling time: 1 business day
- Shipping: tracked service only
- Returns: 30 days, buyer-paid unless defect

## Description structure
1. What problem it solves (1 line)
2. Key specs (material / size / qty)
3. What's included
4. Shipping & return promise
